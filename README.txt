-----------------------------------------------------------------------------------------------------------------------------------
This app is a dominoes ordering app where you can order food from a dominoes store.                                                
the app uses the dominoes API to locate stores nearest you and allow you to order food
i created a GUI to make the ordering easier 
-----------------------------------------------------------------------------------------------------------------------------------
Instructions:
1. Clone the project
2. use the CD command to get into the project 
3. Run pip3 install -r requirements.txt
4. run python3 pizzaGui.py
5. Use app
-----------------------------------------------------------------------------------------------------------------------------------
How is it built?
This app is built in python and i used TKinter to make the GUI. 
i used the API to locate stores and access the dominoes menu and to place the order.
-----------------------------------------------------------------------------------------------------------------------------------
How is it designed?
Every screen is in a class loacted in the screens folder. That is how i made the design for the screens
Each step in the ordering process is in a screen with all the design components.
